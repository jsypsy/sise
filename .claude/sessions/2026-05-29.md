   - 이유: 가장 비파괴적인 방법

2. **탭 네비를 별도 `app/nav.tsx` Client Component로 분리**
   - 이유: `usePathname()`은 Client Hook — `layout.tsx`(Server Component)에서 직접 사용 불가

3. **`.gitignore` 수정 — `.env*` → `.env`, `.env.local`, `.env.*.local`**
   - 이유: `.env.local.example`은 커밋 대상인데 기존 `.env*` 패턴이 이를 차단
   - 변경: example 템플릿은 커밋하되 실제 시크릿 파일만 제외

4. **Legacy anon/service_role JWT 키 사용 (Supabase 신형 sb_publishable_ 키 미사용)**
   - 이유: `@supabase/supabase-js` 기존 호환성 보장, 코드 변경 불필요

5. **seed.ts 정제 로직 인라인 — ingest.ts에서 import 안 함**
   - 이유: CLAUDE.md "Simplicity First" — scripts 간 공유 모듈 추가는 불필요한 추상화
   - 대안 검토: `scripts/lib/shared.ts` → 과도한 추상화로 기각

6. **`process.loadEnvFile('.env.local')` 사용 (dotenv 패키지 미사용)**
   - 이유: Node v20.12+ 내장 기능, 외부 의존성 추가 불필요. 현재 v24.15 환경

### seed.ts 시그널 패턴 설계

- **짝수 그룹 (is_high):** 배율 `[1.00→1.06]` 단조 상승 → 최근 월 3건이 이전 최고가 돌파
- **홀수 그룹 (is_rebound):** 배율 `[1.00, 1.01, 1.02, 1.03, 0.985, 0.965, 0.975]`
  - 최근 월 가격: `base × 0.975`, 전고점: `base × 1.030`
  - 회복률 = 0.975 / 1.030 × 100 = **94.7% ≥ 90%** ✅
  - 직전 대비 상승: 0.975 > 0.965 ✅
- **취소거래:** 매 50번째 트랜잭션 `canceled=true`
- **직거래:** 매 10번째 트랜잭션 `dealing_gbn='직거래'`

> seed는 아직 실행 안 됨 — 다음 세션에서 검증 필요
